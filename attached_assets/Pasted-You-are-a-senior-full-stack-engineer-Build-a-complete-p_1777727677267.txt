PART 1 — BACKEND (FastAPI)
════════════════════════════════════════════

FILE: backend/app.py
- FastAPI with CORS, security headers middleware
- Rate limiting with slowapi
- JWT authentication (RS256)
- All routes prefixed /api/v1
- OpenAPI docs at /docs (dev only)
- Health check at /health

FILE: backend/models/database.py
- SQLAlchemy async with PostgreSQL (Neon.tech)
- Connection string from Replit Secrets: DATABASE_URL
- Tables:
  * users (id, username, argon2_hash, created_at, is_active)
  * devices (id, user_id, name, wireguard_pubkey, created_at)
  * blocklist_entries (id, domain, category, source, added_at)
  * blocked_requests (id, device_id, domain, timestamp, category)
  * threat_reports (id, domain, reporter, votes, verified)
  * firewall_rules (id, device_id, app_package, rule, created_at)

FILE: backend/routers/auth.py
- POST /api/v1/auth/register
  * Body: {username, password}
  * Hash password with Argon2id
  * No email required
  * Return: {user_id, access_token, refresh_token}
- POST /api/v1/auth/login
  * Return JWT (15min expiry) + refresh token (7 days)
- POST /api/v1/auth/refresh
  * Rotate refresh token on use
- POST /api/v1/auth/logout
  * Invalidate refresh token

FILE: backend/routers/vpn.py
- POST /api/v1/vpn/provision
  * Generate WireGuard keypair for device
  * Return .conf file content as string
  * Store public key in devices table
- GET /api/v1/vpn/configs
  * List all user's device configs
- DELETE /api/v1/vpn/revoke/{device_id}
  * Revoke device VPN access
- GET /api/v1/vpn/status
  * Return server stats (connected devices, bandwidth)

WireGuard key generation (pure Python, no system calls):
- Use cryptography library for Curve25519 key generation
- Format as base64 WireGuard-compatible keys
- Generate complete .conf file content

FILE: backend/routers/blocklist.py
- GET /api/v1/blocklist/download
  * Returns compressed (gzip) blocklist
  * 300,000+ domains merged from:
    - StevenBlack hosts (fetch from GitHub)
    - AdAway default list
    - EasyPrivacy
  * Cache in Redis/memory for 24 hours
- GET /api/v1/blocklist/diff?since={timestamp}
  * Returns only new/removed domains since timestamp
- POST /api/v1/blocklist/check
  * Body: {domain}
  * Returns: {blocked: bool, category: string}
- POST /api/v1/blocklist/custom/add
  * User adds custom domain to blocklist
- DELETE /api/v1/blocklist/custom/{domain}
  * Remove from custom blocklist
- GET /api/v1/blocklist/stats
  * Total domains, by category breakdown

FILE: backend/routers/threats.py
- GET /api/v1/threats/feed
  * Latest verified threat domains
  * Paginated, sorted by severity
- POST /api/v1/threats/report
  * Community threat submission
  * Body: {domain, category, description}
  * Rate limited: 10/hour per user
- POST /api/v1/threats/vote/{threat_id}
  * Upvote/downvote threat report
  * Auto-verify at 10+ upvotes
- GET /api/v1/threats/stats
  * Total reports, verified threats, top categories

FILE: backend/services/auth_service.py
- Argon2id hashing:
  * memory_cost=65536, time_cost=3, parallelism=4
- JWT with RS256:
  * Generate RSA keypair on startup (store in Replit Secrets)
  * Access token: 15 minutes
  * Refresh token: 7 days, single-use rotation
- Zero IP logging in all auth functions

FILE: backend/middleware/security.py
Middleware to add these headers to ALL responses:
- Strict-Transport-Security: max-age=31536000
- X-Content-Type-Options: nosniff
- X-Frame-Options: DENY
- X-XSS-Protection: 1; mode=block
- Content-Security-Policy: strict policy
- Referrer-Policy: no-referrer
- Permissions-Policy: camera=(), microphone=(), geolocation=()
Strip all server identification headers.

import subprocess
import threading

def run_backend():
    subprocess.run(["uvicorn", "backend.app:app",
                   "--host", "0.0.0.0", "--port", "8000",
                   "--workers", "2"])

def run_frontend():
    subprocess.run(["npm", "run", "dev", "--prefix", "frontend"])

# Start both
t1 = threading.Thread(target=run_backend)
t2 = threading.Thread(target=run_frontend)
t1.start()
t2.start()
t1.join()
t2.join()
