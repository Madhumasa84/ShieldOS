1. ANDROID DEVICE REGISTRATION FLOW:

Body: {
  "device_name": "My Phone",
  "android_version": "14",
  "app_version": "1.0.0",
  "public_key": "wireguard_public_key_base64"
}
Response: {
  "device_id": "uuid",
  "device_token": "sk_device_xxxxx",
  "wireguard_config": "full .conf file content",
  "blocklist_url": "/api/v1/android/blocklist",
  "dns_endpoint": "/api/v1/dns/query",
  "sync_interval_hours": 24
}

2. ANDROID BLOCKLIST SYNC:
GET /api/v1/android/blocklist
Headers: Authorization: Bearer {device_token}
- Returns gzip-compressed plain text
- One domain per line
- ETag header for cache validation
- Returns 304 Not Modified if unchanged
- Optimized for mobile (no JSON overhead)

Merge into dashboard stats.

4. ANDROID HEARTBEAT:
POST /api/v1/android/heartbeat
Body: { "device_id": "uuid", "vpn_active": true }
- Updates device last_seen
- Returns: { "blocklist_updated": false, "force_sync": false }
- If blocklist updated since last sync → force_sync: true
- Called every 5 minutes by Android app
